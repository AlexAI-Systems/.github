## About

**AlexAI Systems** is currently focused exclusively on building **TermiGram**.

TermiGram is a Telegram Mini App for developers, system administrators, and technical operators who need practical access to their own servers from a mobile-friendly interface. It combines real interactive SSH sessions, SFTP file management, reusable server profiles, controlled sharing, credential protection, command snippets, and AI-assisted technical guidance.

## TermiGram

- [**Interactive SSH terminal**](https://docs.termigram.com/en/docs/terminal/) — work in real shell sessions with tabs, mobile keys, clipboard controls, and notifications.
- [**SFTP file manager**](https://docs.termigram.com/en/docs/sftp/) — browse directories and upload, download, create, edit, rename, and remove remote files without leaving Telegram.
- [**Server profiles**](https://docs.termigram.com/en/docs/server-profiles/) — save, search, check, organize, and reuse server connections.
- [**Command snippets**](https://docs.termigram.com/en/docs/snippets/) — save reusable commands and send them to the active terminal.
- [**Controlled Share Links**](https://docs.termigram.com/en/docs/share-links/) — choose revocable Guest access or Full access profile copies with expiry and activation controls.
- [**Master Password**](https://docs.termigram.com/en/docs/master-password/) — add an optional device-side protection layer for saved SSH passwords and private keys.
- [**SSH keys and import/export**](https://docs.termigram.com/en/docs/ssh-keys/) — generate keys, use existing credentials, parse SSH config, and transfer profiles carefully.
- [**AI assistant**](https://docs.termigram.com/en/docs/ai-assistant/) — get guidance on TermiGram, SSH, Linux, and terminal workflows through the main bot.
