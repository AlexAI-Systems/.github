# AlexAI Systems

**AlexAI Systems** разрабатывает AI-assisted developer tools и прикладные
продукты для инженерных команд, которым важны скорость, безопасность и
понятный production-процесс.

Текущий основной продукт - **TermiGram**: Telegram Mini App для SSH-терминала и
SFTP-доступа. Продукт помогает подключаться к своим серверам из Telegram,
работать в shell, управлять файлами и выполнять частые DevOps-сценарии без
отдельного desktop-клиента.

## Направления работы

- Developer tools для быстрых инфраструктурных сценариев.
- Telegram-native интерфейсы для рабочих процессов разработчиков и операторов.
- AI-assisted engineering: прозрачная связка задач, кода, проверок и релизов.
- Security-first подход к хранению секретов, доступам и production-практикам.

## TermiGram

TermiGram объединяет несколько рабочих сценариев в одном Mini App:

- интерактивный SSH-терминал;
- SFTP-навигация, загрузка, скачивание и редактирование файлов;
- сохраненные подключения с шифрованием секретов;
- snippets для часто используемых команд;
- безопасные deep-link/share сценарии.

Публичная точка входа: [termigram.com](https://www.termigram.com/)

## Принципы

- **Надежность важнее демонстрационного эффекта.** Сначала стабильный рабочий
  сценарий, затем расширение возможностей.
- **Безопасность по умолчанию.** Секреты, ключи, токены и приватные данные не
  публикуются в репозиториях, issue, логах и документации.
- **Проверяемый delivery.** Существенные изменения проходят через задачу,
  отдельную ветку, проверку, staging и production-контроль.
- **Русскоязычная проектная коммуникация.** Описания, задачи, PR и handoff notes
  ведутся на русском; команды, API names, paths и technical identifiers
  остаются в оригинале.

## Статус

AlexAI Systems находится на этапе активной разработки продуктовой и инженерной
основы. Часть репозиториев и production-деталей остается приватной до
завершения соответствующих security и product readiness этапов.
